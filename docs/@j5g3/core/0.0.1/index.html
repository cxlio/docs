Installation

Key features include:

  • Uses WebGPU API for high-performance GPU rendering.
  • Instanced rendering with dynamic per-instance model matrices and colors.
  • Shader-based transformations and texture sampling using WGSL.
  • Efficient management of GPU buffers and resources.
  • Designed in TypeScript with strong typing.

Visit the demo to see the engine in action.
